# Pipeline Integrity Dashboard

## Project Overview
This project analyzes pipeline integrity data to assess material performance, thickness degradation, and overall pipe condition. It includes data cleaning, KPI calculation, and a dashboard visualization.

## Key Features
- Cleaned and validated raw pipeline data
- Calculated KPIs:
  - % Pipes in Poor Condition
  - Average Thickness Loss (mm)
  - Average Thickness Loss Rate (mm/year)
  - Highest Risk Material
- Interactive Excel dashboard with slicers
- Pivot charts for data visualization
- PDF & PNG snapshots for easy viewing on any device

## Files Included
- `cleaned_pipeline_data.xlsx` → cleaned dataset used for analysis
- `Pipeline_Integrity_Dashboard.pdf` → full dashboard (mobile-friendly)
- `dashboard_overview.png` → dashboard snapshot

## How to Use
- Open the Excel file to interact with the dashboard and slicers
- Use PDF or PNG if viewing on mobile or for quick review
